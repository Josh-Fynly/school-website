# Banner of Excellence Schools Website

A modern, Apple-inspired website for Banner of Excellence Schools - providing quality early childhood education.

## Features

- **Responsive Design** - Works seamlessly on all devices
- **Apple-Inspired** - Minimalist, clean, and professional
- **Dynamic Content** - Easy to update and customize
- **Fast Performance** - Built with Next.js and optimized
- **Professional UI** - Pink and navy color scheme matching school branding

## Pages

- **Home** - Hero section with program overview
- **About** - School mission, vision, and values
- **Programs** - Details on Crèche, Nursery, and Primary programs
- **Gallery** - Photo gallery of school activities
- **Contact** - Enrollment inquiry form

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Vercel (Deployment)

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Josh-Fynly/school-website.git
cd school-website

# Install dependencies
npm install

# Run development server
npm run dev
Visit http://localhost:3000 to see the website.
Building for Production
npm run build
npm start
Deployment
This website is deployed on Vercel. To deploy:
Push code to GitHub
Connect repo to Vercel
Vercel auto-deploys on push
Customization
Colors
Edit app/globals.css to change:
--primary-pink: Main color
--navy: Accent color
Content
Edit individual page files in app/ directory:
app/page.tsx - Homepage
app/about/page.tsx - About page
app/programs/page.tsx - Programs page
app/gallery/page.tsx - Gallery page
app/contact/page.tsx - Contact/Enrollment form
Adding Logo
Place logo in public/images/logo.png
Import in components/Navbar.tsx
Replace text logo with image
